**Keyboard Simple** is a lightweight Android keyboard library built with Java and Kotlin,
designed to be dropped into any Android project with minimal configuration.
It is distributed via [JitPack](https://jitpack.io) and covers the most common keyboard
customization needs without pulling in heavy dependencies.

## Integration

### 1. Add JitPack to your repositories

```kotlin
// settings.gradle.kts (or build.gradle)
dependencyResolutionManagement {
    repositories {
        maven { url = uri("https://jitpack.io") }
    }
}
```

### 2. Add the dependency

```kotlin
// build.gradle.kts (app module)
dependencies {
    implementation("com.github.NearApps:Keyboard-Simple:1.0.5")
}
```
