**Keyboard Simple** es una librería Android de teclado construida con Java y Kotlin,
diseñada para integrarse en cualquier proyecto Android con configuración mínima.
Se distribuye via [JitPack](https://jitpack.io) y cubre las necesidades más comunes
de personalización de teclado sin dependencias pesadas.

## Integración

### 1. Agrega JitPack a tus repositorios

```kotlin
// settings.gradle.kts (o build.gradle)
dependencyResolutionManagement {
    repositories {
        maven { url = uri("https://jitpack.io") }
    }
}
```

### 2. Agrega la dependencia

```kotlin
// build.gradle.kts (módulo app)
dependencies {
    implementation("com.github.NearApps:Keyboard-Simple:1.0.5")
}
```
